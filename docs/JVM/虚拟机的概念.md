## 1 什么是虚拟机

### 1.1 JVM 的定义

* jvm 是一种规范
  * [Java Language and Virutual Machine Specifications](https://docs.oracle.com/javase/specs/index.html)
* 虚构出来的一台计算机
  * 字节码指令集（汇编语言）
  * 内存管理：栈 堆 方法区等
